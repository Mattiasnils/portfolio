The Secret Message 5

2015

Description

The goal of this project is simple: input the correct word into the dialogue box in order to obtain the secret message. Sounds easy? Think again.

The Secret Message 5 is the most advanced in a series of projects designed to be hacked. The idea is that players have full access to the source code and can modify it however they like to force the project to decrypt the secret message.

However, the source code itself is architected in a way that causes maximum confusion and there are measures in place that i.e. detects attempts to brute force the correct word.

The message, in turn, takes the player on a journey across multiple Scratch projects, each containing a challenge and lead that points to the next project. I collaborated with 7 other prominent "Scratchers" to have them place secret code within their projects.
